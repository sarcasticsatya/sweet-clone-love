# Nythic AI - Karnataka SSLC EdTech Platform

AI-powered learning platform for Karnataka SSLC Kannada-medium students. Interactive study materials, AI chat, quizzes, and flashcards.

## Technologies Used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

```sh
# Install dependencies
npm install

# Start development server
npm run dev
```

## Features

- AI-powered chat with curriculum-bound responses
- Automatic flashcard generation
- Quiz generation with scoring
- Mind map visualization
- Video library
- Multi-language support (Kannada & English)
- Admin dashboard for content management
- Student performance tracking
